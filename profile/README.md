### Pulse - AI Operator OS

**The agent infrastructure Vaylo runs on - and deploys for clients.**

A NeuralOS workspace overlay that turns every Claude Code session into one canvas. Your business, running itself - autonomous lead gen, outreach, CRM, memory cortex, and workflow execution. Deployed on client infrastructure in a 2-week sprint.

**Architecture**
- **Pulse Hub** - shared state, agent registry, run history, telemetry
- **NeuralOS Overlay** - live canvas visualizing every agent and strand in real time
- **Memory Cortex** - SOPs, context, institutional memory - executable in one click
- **Strand Execution** - high-leverage workflows captured as reusable one-click automations
- **Mobile Companion** - monitor and trigger from anywhere
- **Daily AI Digest** - your business, summarized every morning
